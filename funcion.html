FUNCION DE LOS SERVIDORES WEB

El protocolo utilizado para la transmisión de archivos es HTTP o HTTPS (ofrece un nivel extra de seguridad gracias al cifrado). Este protocolo se basa a su vez en los protocolos de red IP y TCP. Así, un servidor web puede mostrar el contenido de un sitio web de forma simultánea a varios navegadores web. Es importante escoger el servidor web que mejor se adapte a las necesidades de cada persona o empresa. El principal objetivo es evitar sobrecargas en el servidor que puedan ralentizar la velocidad de carga del sitio web.
